HTML

  • HTML (Hypertext Markup Language) is one of the three cornerstone languages used in web development.
  • HTML is used to create the structure of a webpage.
  • It is made up of elements which work together to provide context and the basic framework of all pages on the web.
  • Elements is a type of content on a web page
  • The following elements are used to add content pages
  • 1. Text elements
  • 2. Image elements
  • 3. Header elements
  • 4. List elements
  • Element types divides the content on the page and describes the content to browsers and other developers.
  • Common element types include:
  • 1. Meta-related elements
  • 2. Content division elements
  • 3. Semantic elements
  • HTML Elements work together to build the strucutre of a webpage and provide a framework that makes it easy for a user to navigate the webpage.

  • HTML Elements are typically composed of opening and closing tags which are represent with "angle brackets"
  • HTML Documents structure contains
  • 1. Header Elements: contains information or metadata that helps the browser to render the page correctly. Elements included within the header are not visible when are user looks at a webpage.
  • 2. Body Elements: Contains the content that is shown to the user. All the elements that we want the user to see and interact with must be places inside this element.
